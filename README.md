## Запуск
1. Разархивировать data/dialogues.txt.zip:
```
unzip data/dialogues.txt.zip -d data/
```

2. Создать `.env` из `example.env`
```
cp example.env .env
```

3. Запустить docker контейнер
```
docker-compose down && docker-compose up --build
```
